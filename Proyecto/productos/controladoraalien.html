Controladora Alien

La Controladora de Bombas de Combustible es un sistema centralizado que gestiona el funcionamiento de múltiples bombas de combustible en una estación. Esta placa de control se encarga de coordinar el flujo de combustible, la emisión de recibos, el monitoreo de los niveles de combustible y la comunicación con otros sistemas como impresoras y pantallas, garantizando una operación eficiente y segura.

Con su capacidad para gestionar varias bombas al mismo tiempo, la Controladora asegura que cada transacción y operación se realice de manera sincronizada, permitiendo un control completo y optimizado de la estación de servicio.

Compatibilidad:
Varias bombas de combustible, pantallas, impresoras
Puertos:
USB, Ethernet, Conexión directa a bombas
Control:
Gestión de múltiples bombas, monitoreo de flujo, emisión de recibos
Alimentación:
12V – 24V
Funciones:
Control de varias bombas, supervisión en tiempo real, integración con otros dispositivos
Precio:
8,599 Bs.
